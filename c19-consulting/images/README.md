# Images — mode d'emploi

Dépose tes photos ici avec **exactement** ces noms de fichiers (en .jpg, minuscules).
Dès qu'un fichier existe, il apparaît sur le site. S'il manque, la planche grise s'affiche à la place. Rien d'autre à faire.

Chaque image sert à la fois pour la vignette (grille) et la fiche projet.

**Format conseillé** : portrait 3:4 (ex. 1200 × 1600 px), JPG de qualité 80 environ, moins de 500 Ko par image pour que le site reste rapide. Tu peux exporter ça depuis Photos/Lightroom ou compresser sur squoosh.app.
